This repository contains selected C# scripts from **Meows of Wrath**, a Unity 2D platformer prototype.
The project focuses on building gameplay systems and a responsive character controller (rather than shipping a full game).

**Playable demo:** https://mthierry64.itch.io/meows-of-wrath

---

## Highlights

- Custom character controller & movement logic (C#)
- Raycast-based collision handling (no reliance on Rigidbody physics for core movement)
- Ladders / vertical movement logic
- Secret wall / reveal mechanics
- Basic checkpoint / respawn game state support
- Pause + audio control hooks
